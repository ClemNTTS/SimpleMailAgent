Configuration IMAP

Pour commencer à utiliser EmailManager, veuillez configurer vos paramètres IMAP. Assurez-vous d'avoir activé l'authentification à deux facteurs sur votre compte Gmail et d'avoir généré un mot de passe d'application.

Votre adresse email Gmail
Le mot de passe d'application généré dans les paramètres de sécurité Google
Le serveur IMAP de votre fournisseur de messagerie (par défaut : imap.gmail.com)
Votre clé API Google Gemini (optionnel)
